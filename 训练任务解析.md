# 训练任务

## 1. 使用 DAC 通道 A 发生一个 100Hz 的正弦波

使用 DAC 通道 A 发生一个 100Hz 的正弦波，更新频率 20kHz，峰峰值为 1.65V，偏置为 1.65V。

> 注：发生正弦波需要使用 `ctl_sin()` 函数。

### 分析

由 **5.1.2 DAC 输出** 部分可知，其需要在 `output_callback` 中完成，可定位到 `xplt.ctl_interface.h`，垂直分辨率为 12bit（4096），格式为：

```c
DAC_setShadowValue(IRIS_DACA_BASE,
                   (uint16_t)(adc_v_out.control_port.value * 2048.0f + 2048.0f));
```

其中，`IRIS_DACA_BASE` 代表 DAC 的 A 通道，第二个参数包括三个部分，要注意其输出的为标幺值，因此需要进行转换，因为输入为 [-1, 1]，即 `* 2048.0f` 代表等比例缩放（恰好覆盖 4096），最后 `+ 2048` 的作用相当于给 DAC 加了一个中点偏置。

了解了函数的具体形式，我们还需知道电源电压/DAC 满量程是 3.3V，但题目要求输出的正弦波不是满量程正弦，而是峰峰值 1.65V、偏置 1.65V，因此实际 DAC 输出电压应该是：

$$
V_{out} = 1.65 + 0.825\sin\theta
$$

换成 DAC 数字量：

$$
DAC_{code} = \frac{1.65 + 0.825\sin\omega t}{3.3}\times4096 = 2048 + 1024\sin\omega t
$$

因此实际代码为：

```c
GMP_STATIC_INLINE void ctl_output_callback(void)
{
    static uint32_t tick = 0;

    tick += 1;

    ctrl_gt output_signal = (tick % 200) / 200.0f;

    DAC_setShadowValue(IRIS_DACA_BASE, ctl_sin(output_signal) * 1024 + 2048);
}
```

## 2. 使用 ADC 采集这个正弦波并标幺

使用 ADC 采集这个正弦波，并且减去 1.65V 偏置、以 1.65V/2 为单位 1 进行标幺。

> 注：正弦波的标幺可以使用 `adc_channel_t` 类型提供的功能。
>
> 参考头文件：`gmp_pro\ctl\component\interface\adc_channel.h`

### 分析

由 **5.1.3 ADC 输入** 部分，需要在 `input_callback` 中进行操作，垂直分辨率 12bit。

已知：

```c
#define ADC_CH1_ADC_SOC_NUMBER ADC_SOC_NUMBER0
#define ADC_CH1_RESULT_BASE    ADCARESULT_BASE
#define ADC_CH1_ADC_BASE       ADCA_BASE
#define ADC_CH1_CHANNEL        ADC_CH_ADCIN6
```

读取 ADC 结果时，应该主要用：

```c
ADC_CH1_RESULT_BASE
ADC_CH1
```

即：

```c
ADC_readResult(ADC_CH1_RESULT_BASE, ADC_CH1_ADC_SOC_NUMBER);
```

代码为：

```c
static adc_channel_t adc_v_in;

// Input Callback
GMP_STATIC_INLINE void ctl_input_callback(void)
{
    ctl_step_adc_channel(&adc_v_in, ADC_readResult(ADC_CH1_RESULT_BASE, ADC_CH1));
}
```

## 3. 使用超前补偿器进行超前补偿

使用超前补偿器对于标幺后的信号做超前补偿，超前补偿器的补偿频率设置为 100Hz，超前角度设置为 45°（π/4），增益为 1。

> 注：超前补偿器可以使用 `ctl_lead_t` 模块。
>
> 参考头文件：`gmp_pro\ctl\component\intrinsic\discrete\lead_lag.h`

---

## 4. 通过 DAC 通道 B 输出补偿后的波形

将经过超前补偿得到的波形通过 DAC 通道 B 输出，更新频率仍然为 20kHz。

---

## 5. 通过 PWM 输出补偿后的波形

将控制器输出的补偿后的波形标幺在 [0, 1] 区间内，通过 PWM 调制发生在 IRIS_EPWM1 通道，死区设置为 500ns，死区极性设置为低电平（即 EPWM1A 和 EPWM1B 在死区时间同时输出低电平）。

> 注：PWM 调制器的输出可以使用 `pwm_channel_t` 模块。
>
> 参考头文件：`gmp_pro\ctl\component\interface\pwm_channel.h`
